# Solworld zh_cn patch skeleton

Create per-mod language files here:

`assets/<modid>/lang/zh_cn.json`

Example:

`assets/examplemod/lang/zh_cn.json`

```json
{
  "item.examplemod.example_item": "Example Name",
  "block.examplemod.example_block": "Example Block"
}
```

Tips:

- Keep only missing or override keys.
- Use Language Reload to hot-reload while testing.
- This pack should stay at high priority.
